# Telegram Bot with Flask API

这个项目演示了如何创建一个Telegram机器人，该机器人集成了Flask作为Web服务器，并且可以与MySQL数据库交互。它可以处理基本的Telegram命令、回显消息，并通过Flask处理HTTP请求。

## 功能

- 响应Telegram命令
- 回显用户消息
- 接收HTTP请求
- 记录数据到MySQL数据库

## 开始

以下指南将帮助你在本地机器上安装和运行项目，用于开发和测试目的。

### 先决条件

- Python 3.6+
- pip
- MySQL数据库
